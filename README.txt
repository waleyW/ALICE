pdf_count.py is used to count the number of PDF files
search_case.py is used to search for queries with case sensitivity
search_nocase.py is used to search for queries without case sensitivity

How to use this script:
1. Install Python from https://www.python.org/downloads/
2. Edit the queries below:
   - For AND relations: Put words in the same list
   - For OR relations: Put words in separate lists
3. Run in command line: python search_nocase.py
